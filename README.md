# Quiz Application

Welcome to the Quiz Application! This project is a simple quiz application built using Python and Tkinter. The application allows users to select a category and difficulty level, and then proceed to answer a series of quiz questions.

## Features

- Introductory page to select quiz category and difficulty
- Interactive quiz interface with multiple-choice questions
- Real-time score display
- Timer to limit the duration of each quiz
